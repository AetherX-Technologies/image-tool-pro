# 快速启动指南

## 环境要求

- Python 3.7+
- Windows/Linux/MacOS

## 安装步骤

### 1. 安装依赖

```bash
pip install pillow
```

或者使用requirements.txt：

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python main.py
```

## 快速测试

### 测试核心功能

运行测试脚本验证所有核心功能是否正常：

```bash
python test_core.py
```

如果看到 `✓ 所有测试通过！`，说明安装成功！

## 第一次使用

1. **启动应用**
   ```bash
   python main.py
   ```

2. **打开图片**
   - 点击菜单 "文件" → "打开图片"
   - 或按快捷键 `Ctrl+O`

3. **尝试裁剪**
   - 在图像上左键拖动绘制红色裁剪框
   - 拖动四角的红点调整大小

4. **保存结果**
   - 点击 "保存图片" 按钮
   - 或按快捷键 `Ctrl+S`

## 主要功能

### 1. 交互式裁剪
- **操作**: 左键拖动绘制裁剪框
- **调整**: 拖动四角红点
- **查看**: 右侧显示选区像素值

### 2. 中心点切割
- **设置中心**: 右键点击图像
- **输入尺寸**: 在右侧输入宽度和高度
- **执行**: 点击"执行切割"按钮

### 3. 图像压缩
- **目标大小**: 输入数字（如 500）
- **选择单位**: KB 或 MB
- **选择格式**: JPEG 或 PNG
- **执行**: 点击"开始压缩"按钮

### 4. 预览对比
- **操作**: 点击"预览对比"按钮
- **效果**: 并排显示原图和处理后的图像

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 打开图片 |
| `Ctrl+S` | 保存图片 |
| 左键拖动 | 绘制/调整裁剪框 |
| 右键点击 | 设置中心点 |

## 故障排除

### 问题1: 无法启动应用
**解决方法**: 确保安装了Pillow库
```bash
pip install pillow
```

### 问题2: 打开图片失败
**解决方法**: 确保图片格式为 JPG, PNG, BMP 或 GIF

### 问题3: 压缩后文件大小不精确
**解决方法**: 这是正常现象，图像压缩是非线性的，程序会尽量逼近目标大小

## 示例工作流

### 场景1: 裁剪头像
1. 打开图片（Ctrl+O）
2. 左键拖动绘制正方形裁剪框
3. 调整到合适的大小和位置
4. 保存图片（Ctrl+S）

### 场景2: 制作缩略图
1. 打开图片
2. 在右侧输入目标大小（如 100KB）
3. 选择JPEG格式
4. 点击"开始压缩"
5. 保存图片

### 场景3: 中心裁剪
1. 打开图片
2. 右键点击图片设置中心点（或留空使用默认中心）
3. 在右侧输入宽度和高度（如 800 x 600）
4. 点击"执行切割"
5. 保存图片
